# A Ray Tracer in Rust

A Rust implementation of the ray tracer from Peter Shirley's [Ray Tracing in One Weekend](https://www.amazon.com/gp/product/B01B5AODD8).

## Usage

```sh
$ cargo run --release -- --help
```
